# App_Data_AB_Testing
Расчет параметров A/B-теста, который позволит проверить, повысят ли изменения вовлеченность пользователей, и анализ его результатов.


### Описание данных

Для анализа используются три таблицы:

| Таблица | Описание |
|----------|----------|
| `sessions_project_history.csv` | Исторические данные по сессиям пользователей за период с `2025-08-15` по `2025-09-23` |
| `sessions_project_test_part.csv` | Данные за первый день проведения A/B-теста (`2025-10-14`) |
| `sessions_project_test.csv` | Данные за весь период проведения A/B-теста с `2025-10-14` по `2025-11-02` |

> Таблицы имеют практически одинаковую структуру и отличаются только периодом наблюдения.

---

### Структура данных

Таблицы `sessions_project_history.csv`, `sessions_project_test.csv` и `sessions_project_test_part.csv` содержат следующие поля:

| Поле | Описание |
|--------|----------|
| `user_id` | Идентификатор пользователя |
| `session_id` | Идентификатор сессии в приложении |
| `session_date` | Дата сессии |
| `session_start_ts` | Дата и время начала сессии |
| `install_date` | Дата установки приложения |
| `session_number` | Порядковый номер сессии для конкретного пользователя |
| `registration_flag` | Признак регистрации пользователя |
| `page_counter` | Количество просмотренных страниц во время сессии |
| `region` | Регион пользователя |
| `device` | Тип устройства пользователя |
| `test_group` | Тестовая группа пользователя (отсутствует в таблице `sessions_project_history.csv`) |

---

### Особенности данных

- Таблица `sessions_project_history.csv` содержит только исторические данные и **не включает** столбец `test_group`.
- Таблицы `sessions_project_test_part.csv` и `sessions_project_test.csv` содержат данные A/B-теста и включают информацию о принадлежности пользователей к тестовым группам.
- Каждая строка соответствует одной пользовательской сессии.
